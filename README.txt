== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `mk-todo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit todo list in 'MK ToDo Settings' menu in Admin area.
4a. Place `<?php do_shortcode('mk-todo'); ?>` in your template or `[mk-todo]`
   in edited content.
4b. Select widget through 'Widgets' submenu in 'Appearance' menu.
